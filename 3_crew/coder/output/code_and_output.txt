```python
# This program calculates the first 10,000 terms of the series:
# 1 - 1/3 + 1/5 - 1/7 + ... and multiplies the result by 4.

def calculate_series(terms):
    total = 0.0
    for n in range(terms):
        if n % 2 == 0:
            total += 1 / (2 * n + 1)  # positive term
        else:
            total -= 1 / (2 * n + 1)  # negative term
    return total * 4

# Number of terms to calculate
num_terms = 10000
result = calculate_series(num_terms)

# Output the result
print(f"The result of the series multiplied by 4 for the first {num_terms} terms is: {result}")
```

Output of the code:
```
The result of the series multiplied by 4 for the first 10000 terms is: 3.1415926535897743
```

This output provides the approximation of Pi based on the provided series calculation, confirming that the algorithm correctly computes the series as intended.
